# DAEMON-POLY Bot — Replit Setup
# 5 steps, under 10 minutes

## STEP 1 — Create Replit Account
Go to replit.com and sign up (free)

## STEP 2 — Create New Repl
Click + Create Repl
Select Python
Name it: daemon-poly-bot
Click Create

## STEP 3 — Upload the Bot File
Delete the default main.py content
Paste the entire contents of daemon_poly_bot.py into main.py

## STEP 4 — Add Your Secrets (click the padlock icon in left sidebar)
Add each of these as a Secret — key name on left, your value on right:

SLASH_GOLF_KEY        → your RapidAPI key from rapidapi.com dashboard
TELEGRAM_TOKEN        → your new bot token from BotFather
POLYMARKET_API_KEY    → your Polymarket API key
POLYMARKET_PK         → your Polymarket wallet private key
POLYMARKET_WALLET     → your Polymarket wallet address (0x...)
ANTHROPIC_API_KEY     → your Claude API key from console.anthropic.com
SESSION_BANKROLL      → your starting amount e.g. 300

TELEGRAM_CHAT_ID is already hardcoded as 8486909237 — no need to add this

## STEP 5 — Run
Click the green Run button
Bot will start and send you a Telegram message confirming it's online

Then in Telegram send:
BANKROLL: $300       (or whatever amount you're deploying)
AUTOPILOT: ON        (activates trade execution)

## KEEP IT RUNNING
Free Replit repls sleep after inactivity.
Go to Replit sidebar → expand your repl → turn on Always On ($7/mo)
OR keep the browser tab open on your phone all weekend

## COMMANDS (send these to your Telegram bot anytime)
AUTOPILOT: ON         activate trading
AUTOPILOT: OFF        signals only, no trades
AUTOPILOT: RESUME     restart after suspension
STATUS                see open positions and P&L
SCAN                  run manual scan
BANKROLL: $[X]        update bankroll
COOLDOWN: RESET       clear cooldown after 3 losses
CYCLE PROFIT          manually trigger profit cycling
CYCLE: OFF            disable cycling
EXIT HALF [player]    sell half a position
EXIT FULL [player]    close a full position
